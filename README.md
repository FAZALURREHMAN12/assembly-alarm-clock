# Assembly Language Alarm Clock

### Overview
A low-level hardware interaction project written in x86 Assembly language. This program functions as a real-time alarm clock, demonstrating an understanding of system architecture by directly interacting with BIOS and hardware interrupts.

### Technologies Used
* x86 Assembly Language (emu8086)
* BIOS Interrupts

### Key Features
* **Low-Level Hardware Control:** Engineered precise delay logic and time-tracking utilizing BIOS interrupts.
* **Direct Input/Output:** Handled user input directly via keyboard interrupts and managed real-time display updates within limited screen space.
* **Optimization:** Fine-tuned execution flow and memory usage to ensure accurate, real-time CPU clock cycles.
